# Muse
Muse is a browser, forked from this repro: https://github.com/htoukour/Mozart

# Features and Stuff I will add

Muse has:

* Tabs
* Reload
* Back and Forward
* Print file as
* Saving Webpages
* Opening HTML files

What I want to add:

* Adblock
* Bookmarks
* Shortcuts
* Password Manager
* Bongo, the music player. (Work in Progress)
